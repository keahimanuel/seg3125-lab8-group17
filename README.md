## Description

Paws & Home is a React Web Application designed to help users browse adorable pets, view detailed pet profiles, learn about pet care, communicate with the shelter and complete an adoption application. 

This project is based on the design created in **Lab 7**.

## Human Interactive Processes

The application focuses on these 4 human interactive processes: 

- **Exploring** - users browse available pets using search, filters and sorting
- **Absorbing information** - users read pet details, care guides and medical info
- **Following instructions** - users complete a multi-step adoption form
- **Communicating** - users interact through the messaging page

## Installation

1. Open the project folder in terminal
2. In Terminal, run : 
```[ssh]
npm install
npm install react-router-dom
npm run dev
```
3. Open the app in your browser
